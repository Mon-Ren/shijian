🩺 AI 的通病

当前的 AI Agent 会思考，但不会"想问题"。
🌀

抓不住重点

面对复杂问题，胡子眉毛一把抓，分不清主次。

🗣️

不调查就下结论

信息不充分就急着给答案，犯教条主义错误。

😴

做完不自查

"差不多就行了"，缺乏验证和复盘的习惯。

🏳️

遇难即退

第一次碰壁就说"超出能力范围"，缺乏持续推进力。

🎯

同时做十件事

什么都想做，件件做不好，不懂集中兵力。

📋

只会执行不会想

用户说什么就做什么，没有自己的分析和判断。
